AI-First Operating Model

Agents run our team. So we can run yours.

113S is built AI-first. Agents handle the internal work that traditional advisory firms burn analyst hours on — research, screening, diligence prep, market mapping, deal triage — so our partners spend their time on judgment, relationships and execution. We bring the same operating model to the mandates we run for you.
